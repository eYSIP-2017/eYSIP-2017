These Videos show the Rendezvous of two robots without communicating with each other, but by only sensing the surrounding for any other robots and going to the centroid of the shape formed by these robots. (AKA Rendezvous Problem)

Platform : Firebird V (Nex Robotics)
For further reference : https://github.com/akshar100/eyantra-firebird-resources/tree/master/Fire%20Bird%20V%20ATMEGA2560%20Robots%202010-12-29

Code for the same can be found @ https://github.com/eYSIP-2017/eYSIP-2017_DistributedRobotics/tree/master/Codes/2017-5-31/20170530_1452_RandevousProblem
